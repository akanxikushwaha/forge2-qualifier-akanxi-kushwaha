# Forge 2 Qualifier — Kanban App

## What it does
A Trello-style Kanban board with Boards, Lists, Cards, Tags, Members and due dates.

## Models used
- OpenClaw (hands): google/gemini-2.5-flash with groq/llama-3.1-8b-instant fallback
- Hermes (brain): gemini-2.5-flash

## Why this routing
Hermes uses Gemini for large context planning. OpenClaw uses Gemini/Groq for fast code execution.
